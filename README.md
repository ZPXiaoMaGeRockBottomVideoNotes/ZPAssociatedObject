# ZPAssociatedObject
本Demo主要介绍了原类与分类在.h文件中添加属性之后的不同之处。还介绍了分类能够添加属性，但是不能添加成员变量。还从两方面解释了给分类添加属性后，系统为什么不能像原类一样生成一个带下横线的成员变量。还介绍了在分类中添加完属性之后，使用Runtime运行时机制的关联对象方式重写属性的set和get方法，使之在其他类中能够正常使用点语法来给分类的属性赋值和取值。
